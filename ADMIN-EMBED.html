Emergency Response System - Quick Reference Guide

🏠 PUBLIC PORTAL 🔐 ADMIN DASHBOARD 📊 DATA SPREADSHEET

📋 Daily Routine (5 minutes each morning)

  1. Check email for emergency alerts (🚨 Emergency emails)
  2. Open admin dashboard for new requests
  3. Check spreadsheet for high-priority needs
  4. Update maps with new addresses (daily mapping)

🚨 Emergency Alert Response

Emergency Email Format:
🚨 Emergency Priority: Medical Request
Name: John Smith
Phone: 555-123-4567 (← Click to call)
Address: 123 Main St, Tampa, FL 33602 (← Click for directions)
Details: Diabetic emergency, needs insulin

Immediate Actions:

  1. Call phone number immediately
  2. Send help or connect with resources
  3. Update spreadsheet as "handled"

🗺️ Daily Mapping (10 minutes)

  1. Open the spreadsheet (link above)
  2. Look for "📍 Mapping Tools" menu at the top
  3. Click "🗺️ Export for Map"
  4. Download the file it creates
  5. Upload to your map to show where help is needed

🚨 5-Minute Emergency Setup

If you need this system running RIGHT NOW:

  1. Copy the spreadsheet (File → Make a Copy)
  2. Update email in Apps Script (Extensions → Apps Script)
  3. Share form links with the public

🎯 You're now operational!

🛠️ Quick Troubleshooting

No Emergency Alerts?

  • Check email in Apps Script
  • Look in spam folder
  • Test with word "emergency"

Forms Not Working?

  • Check spreadsheet connection
  • Test form submission
  • Verify sharing permissions

Mapping Issues?

  • Look for "📍 Mapping Tools" menu
  • Check data in spreadsheet
  • Verify map edit permissions

🔒 Privacy Reminder

Admin tools contain private information. Never share addresses or phone numbers publicly. Log out when finished.
